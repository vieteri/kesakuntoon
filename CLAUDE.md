# Project Instructions

## Runtime

This project uses **Bun** as the JavaScript runtime and package manager.

- Use `bun install` instead of `npm install` / `yarn install`
- Use `bun run <script>` instead of `npm run <script>`
- Use `bun add <package>` instead of `npm install <package>`
- Use `bun remove <package>` instead of `npm uninstall <package>`
- The lockfile is `bun.lockb`, not `package-lock.json` or `yarn.lock`
- Do not use `npm`, `yarn`, or `pnpm` commands
